# Center-of-Internet-Security🌐
'🌐**in CIS context**

##Summary

**Framework**: *Provides the overall structure and strategic guidance to manage cybersecurity risks* (**CIS Controls are an example**) 

**Benchmarks**: *Offer precise, actionable technical guidance to configure specific systems securely, forming a baseline for security and compliance*

💠In essence, CIS develops frameworks to show “what to do” in cybersecurity and benchmarks to show “how exactly to do it”, helping organizations secure systems, maintain standards, and reduce cyber risks
